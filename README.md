# Crypto Trader

A paper trading crypto app built with Flutter.
Trade real cryptocurrencies with fake money using
live market data from CoinGecko.

You start with **$10,000** and can buy and sell
any of the top 100 cryptocurrencies by market cap. Your portfolio
is saved on your device so your positions and balance are always
there when you come back.

## Tech Stack

- Flutter
- Riverpod
- Hive
- Dio
- CoinGecko API

## Features

- **Live market data** — top 100 cryptocurrencies with real prices updated every 60 seconds
- **Paper trading** — buy and sell crypto with $10,000 in fake money
- **Portfolio tracking** — see your total value, daily P&L and all your positions
- **Search** — find any coin instantly by name or symbol
- **Persistent storage** — your portfolio is saved on your device and survives app restarts

## Prerequisites

Before you begin make sure you have the following installed:

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (version 3.0 or higher)
- [Git](https://git-scm.com/)
- A device or emulator to run the app on

Not sure if Flutter is installed? Run this in your terminal:

```bash
flutter doctor
```

This will tell you exactly what is and isn't set up.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/T3NTS/crypto-trader.git
cd crypto-trader
```

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Run the app

```bash
flutter run
```
